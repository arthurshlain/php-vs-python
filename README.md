# php-vs-python

## Same syntax

* Max value of array `max($arr)` vs `max(arr)`
* Min value of array `min($arr)` vs `min(arr)`
* Modulo `$a % $b` vs `a % b`

## Different names of methods

* Split string by character `str_split($str)` vs `list(str)`
